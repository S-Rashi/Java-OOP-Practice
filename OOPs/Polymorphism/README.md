# Polymorphism

This folder contains Java projects demonstrating the Polymorphism principle of Object-Oriented Programming (OOP). Each project uses method overriding to allow different subclasses to provide their own implementation of a common method.

## Projects

# 1. Payment Gateway System

A payment processing system that demonstrates runtime polymorphism through different payment methods.

# Class Hierarchy


Payment
├── CreditCardPayment
├── UPIPayment
└── WalletPayment


Concepts Demonstrated

- Runtime Polymorphism
- Method Overriding
- Dynamic Method Dispatch
- Code Reusability



# 2. Food Delivery System

A food delivery application that calculates delivery charges differently for various restaurant chains.

# Class Hierarchy


Restaurant
├── Dominos
├── McDonalds
└── KFC


Concepts Demonstrated

- Runtime Polymorphism
- Method Overriding
- Dynamic Binding
- Flexible Object-Oriented Design

# Learning Outcomes

- Understand runtime polymorphism in Java
- Implement method overriding
- Use parent class references to invoke subclass methods
- Build flexible and extensible applications using OOP    principles
