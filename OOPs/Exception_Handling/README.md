# 📚 Library Management System (Exception Handling in Java)

A simple Java mini project demonstrating the use of **Exception Handling** through a library book borrowing system.

## Features

- Borrow a book
- Validate book availability
- Handle invalid book names
- Custom exceptions
- Multiple catch blocks
- finally block execution

## Concepts Used

- try-catch-finally
- throw
- throws
- Custom Exceptions
- Exception Propagation
- OOP

## Project Structure

```
LibraryManagementSystem/
│── Main.java
│── Library.java
│── Book.java
│── InvalidBookException.java
│── BookNotAvailableException.java
```

## Learning Outcome

This project demonstrates how Java exception handling can be used to validate business rules and provide meaningful error messages while keeping the application robust and user-friendly.
